# 🚀 Nova Sathi: Core Platform Standards

This skill defines the primary technical and aesthetic standards that MUST be followed for every part of the Nova Sathi application.

## 1. Professional & Aesthetic Standards
*   **Design Aesthetic:** "Cyber-Premium" (Vibrant, glassmorphic, smooth animations, dark-mode first).
*   **Humanised Vocabulary:** DO NOT use spiritual/mystic terms like "Ritual" or "Sanctuary" in the production UI. Use professional terms like **Session**, **Chat**, **Room**, and **Expert**.

## 2. Technical Architecture
*   **Structure:** Mobile-First **PWA** (Installable Progressive Web App).
*   **Frontend:** React (Vite).
*   **Backend:** Node.js, Express, Socket.io.
*   **Database:** Prisma (PostgreSQL primary).
*   **Authentication:** OTP-based login (New/Returning detection).

## 3. Dual Funnel Funnels
Ensure every implementation considers the **Dual Context**:
1.  **Astro & Future:** Career, Marriage, Vastu, Tarot, Numerology. Focus on **Discovery & Rating**.
2.  **Emotional Support/Anonymous Mode:** Breakup, Anxiety, Loneliness. Focus on **Privacy & Calm UI**.

## 4. Admin-Controlled Routing (MVP Model)
*   **Expert Visibility:** Controlled via a Unified Chat Inbox for administrators.
*   **Growth Tools:** Coupons, Free minutes, and Push Notifications must be modular and scalable.

## 5. Deployment & Performance
*   **Hosting:** Client-Side.
*   **Optimization:** High-speed loads and efficient Socket.io state management.
